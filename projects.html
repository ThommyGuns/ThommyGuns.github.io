Eade Fitness Makeover

One of my current projects is remaking the website for Eade Fitness, a gym located in Olean, NY. The current site is old, cluttered and not mobile-friendly. This redesign will be using Twitter Bootstrap in collaboration with the Jekyll Framework. I am also researching CloudCannon, a hosting site and CMS that allows for non-technical users to make changes to the site.
